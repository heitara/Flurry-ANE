What contains each folder?
# android - a version of the classes, which is used when the ANE is part of AIR Android project 
# default - a version of the classes, which is used when the ANE is part of pure AIR project. This is dummy version, which only traces the name of the function and all passed parameters. Added for computability and seamless usage of the AIR simulator.
# src - a version of the classes, which is used when the ANE is part of AIR iOS project
